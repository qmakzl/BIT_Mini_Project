# BIT_Mini_Project
NoSQL과 java를 연동하여 사원정보 검색(이름, 입사년도), 과거 업무이력 검색(사원ID)를 조회하는 프로그램 만들기
최대한 객체지향적 패턴을 이용해서 만들어 봤습니다.
